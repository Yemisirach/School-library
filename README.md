## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

You must have Ruby installed in your computer.

### Clone this repo

`git clone git@github.com:yemisirach/schoool-library.git`

### In your terminal:

`ruby Main.rb`
